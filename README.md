# JDS Segurança Eletrônica 🚀

Site profissional e moderno para **JDS Segurança Eletrônica**, especialista em soluções de segurança eletrônica em Recife e região. O site foi desenvolvido com foco em **SEO**, **conversão de clientes** e **experiência do usuário**.

## 🎯 Sobre a JDS Segurança Eletrônica

A **JDS Segurança Eletrônica** oferece soluções completas em:
- 📹 Câmeras de Segurança (CFTV)
- 🚨 Alarmes Monitorados
- 🔐 Controle de Acesso
- ⚡ Cerca Elétrica e Concertina

Atendemos **Recife**, **Ipojuca**, **Jaboatão dos Guararapes**, **Olinda** e região metropolitana de Pernambuco.

## ✨ Características do Site

### Design Moderno e Responsivo
- Layout profissional com gradientes e animações suaves
- Totalmente responsivo para dispositivos móveis, tablets e desktops
- Navegação intuitiva com menu hamburger em mobile

## 🚀 Como Usar

### Instalação Local
1. Clone o repositório:
   ```bash
   git clone https://github.com/Vitorkr1/jds-seguranca-eletronica.git
   cd jds-seguranca-eletronica
   ```

2. Abra o arquivo `index.html` no seu navegador:
   ```bash
   # Windows
   start index.html
   
   # macOS
   open index.html
   
   # Linux
   xdg-open index.html
   ```
